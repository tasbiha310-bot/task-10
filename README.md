# task-10# Task 10 Sentiment Analysis


## Objective

Analyze client feedback and classify sentiment.


## Dataset

coretech_feedback.csv


## Steps

1. Created feedback dataset.
2. Cleaned text data.
3. Counted sentiments.
4. Created sentiment chart.
5. Found common words.
6. Built sentiment classifier.


## Model

Naive Bayes + CountVectorizer


## Results

The model predicts whether feedback is Positive,
Neutral, or Negative.

Higher accuracy means better sentiment prediction.


## Insights

1. Positive feedback shows good customer satisfaction.

2. Neutral feedback suggests areas for improvement.

3. Negative feedback helps identify service problems.
